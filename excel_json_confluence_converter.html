📖 Как пользоваться точечной нотацией

Точечная нотация позволяет превращать плоские столбцы таблицы в вложенную структуру JSON без написания кода. Достаточно указать путь через точку .

Что ввести в поле "JSON-путь"Как будет выглядеть в JSON
name{ "name": "Иван" }
user.firstName{ "user": { "firstName": "Иван" } }
address.city{ "address": { "city": "Москва" } }
meta.createdBy.name{ "meta": { "createdBy": { "name": "Admin" } } }

Правила:

  • Точки . автоматически создают уровни вложенности.
  • Пробелы и спецсимволы в путях лучше заменять на _ (например: user_first_name).
  • Если несколько столбцов ведут в один объект (например: address.city и address.zip), они объединятся в одну структуру address.
  • Пустые ячейки в Excel станут null или "" в JSON.

💡 Совет: сначала загрузите файл, отредактируйте транслитерированные ключи под себя, затем измените пути через точку там, где нужна вложенность. Нажмите "Сгенерировать" и проверьте предпросмотр.
