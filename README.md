# ✨ Excel → JSON Converter (Confluence Widget)

[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](https://opensource.org/licenses/MIT)
[![Confluence Ready](https://img.shields.io/badge/Confluence-Compatible-blue)](https://www.atlassian.com/software/confluence)

Легковесное веб-приложение для конвертации Excel-таблиц в формат JSON, разработанное специально для работы внутри **Atlassian Confluence** через макрос HTML.

Приложение работает **полностью локально** в браузере пользователя — данные не отправляются на сервер, что гарантирует безопасность и конфиденциальность.

---

## 🚀 Возможности

- 📂 **Локальная обработка** — файл Excel не покидает браузер пользователя
- 🇺➡️🇧 **Умная транслитерация** — автоматическое преобразование кириллических заголовков в латиницу
- 🕸️ **Точечная нотация (Dot Notation)** — создавайте вложенную структуру JSON, просто указывая путь через точку (например, `user.profile.email`)
- ✅ **Выбор данных** — возможность включать/исключать отдельные столбцы таблицы из конвертации
- 💾 **Сохранение пресетов** — экспорт и импорт настроек маппинга столбцов в JSON-файл
- 📖 **Встроенная инструкция** — понятный гайд для пользователя прямо на странице инструмента

---

##  Как внедрить в Confluence

Инструмент не требует установки плагинов или настройки сервера. Всё работает на чистом JS/CSS/HTML.

### Шаг 1. Подготовка библиотеки
Приложение использует библиотеку `SheetJS` для чтения Excel-файлов.
1. Скачайте файл: [xlsx.full.min.js](https://cdn.sheetjs.com/xlsx-latest/package/dist/xlsx.full.min.js)
2. Загрузите этот файл в **Вложения (Attachments)** вашей страницы Confluence.
   > 💡 *Примечание: Если у вас открыт доступ к внешним CDN, можно оставить ссылку на скрипт как есть в коде.*

### Шаг 2. Установка кода
1. Откройте страницу в Confluence в режиме редактирования
2. Добавьте макрос **HTML**
3. Скопируйте код из файла `index.html` и вставьте его в тело макроса
4. **Важно:** Найдите в коде тег `<script src="...">` и убедитесь, что путь ведет на скачанный вами файл из вложений:
   ```html
   <script src="attachment:xlsx.full.min.js"></script>
